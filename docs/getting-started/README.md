Welcome to the Job Referral Network System! This section will help you get up and running quickly.

- [Project Overview](./overview.md) - Understand what this system does

- [Quick Start Guide](./quick-start.md) - Start using the system immediately

## Prerequisites

Before you begin, ensure you have:

- **Node.js** v20+ installed
- **Docker Desktop** running
- **Git** for version control
- **PostgreSQL client** (optional, for direct database access)

## System Requirements

- **Operating System**: Windows, macOS, or Linux
- **Memory**: 4GB RAM minimum (8GB recommended)
- **Storage**: 2GB free space
- **Network**: Internet connection for dependencies
